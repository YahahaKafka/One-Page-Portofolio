Beberapa Projek Hasil NGODING Bersama-sama

Project 1

Game TicTacToe Web

Walaupun game ini hanya bisa dimainkan dalam web, tapi dijamin asik dan gak kalah sama game Tic Tac Toe yang ada di aplikasi. Dengan lawan yang sudah diprogram, membuat kamu semakin tertantang untuk memainkannya.

Project 2

Kalkulator Sederhana

Buat kamu yang suka menghitung, apalagi menghitung hutang teman bisa nih cobain web kalkulator buatan saya. Walaupun sederhana dan berbasis web, kalkulator ini memiliki fungsi yang gak jauh beda sama kalkulator aplikasi maupun kalkulator nyata.

Web sederhana buat kamu yang jarinya gatel pengen scroll terus. Dinamakan Parallax Web Scroll, web ini akan menampilkan sebuah pemandangan dimana kamu harus scroll pemandangan itu untuk melihat keajaiban.

Project 4

Simple Whatsapp Bot

Ini merupakan projek yang membantu kamu untuk melakukan segala aktivitas hanya menggunakan Whatsapp. Bot Whatsapp ini mampu mendownload lagu dari youtube, membuat stiker, bermain game, dan masih banyak lagi.
